# 🎣 PhishGuard — Email Threat Analyser

A cybersecurity tool that analyses emails for phishing indicators using pattern recognition, sender analysis, and social engineering detection. Built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

**Live Demo:** (https://nomcebom-dev.github.io/Phishguard/)

---

## 🔍 What It Does

PhishGuard analyses email content across 6 threat categories:

| Category | What It Detects |
|---|---|
| **Urgency & Fear** | Time pressure, account threat language, false security alerts |
| **Reward Bait** | Prize scams, free offers, lottery/inheritance fraud |
| **Credential Theft** | Requests for passwords, OTPs, banking details, CVV numbers |
| **Sender Analysis** | Suspicious domains, brand impersonation, free email providers |
| **Link Safety** | Raw IP links, URL shorteners, HTTP (insecure), executable files |
| **Impersonation** | SARS, Home Affairs, law enforcement, generic greetings |

---

## ⚙️ Features

- **Boot sequence animation** — simulates a real security tool initialising
- **Threat meter** — animated 0–100% danger score
- **Categorised flags** — each indicator shown with severity (High / Medium / Low)
- **Category breakdown** — individual score bars per threat category
- **Recommendation engine** — actionable advice based on threat level
- **Sample emails** — load a phishing example or a legitimate email to test
- **Fully responsive** — works on mobile and desktop
- **CRT scanline aesthetic** — terminal/hacker visual design

---

## 🛠 Tech Stack

- **HTML5** — semantic structure
- **CSS3** — custom properties, animations, grid layout, responsive design
- **Vanilla JavaScript** — regex-based threat engine, DOM manipulation, IntersectionObserver

No libraries. No build tools. No dependencies. Open `index.html` in any browser.
